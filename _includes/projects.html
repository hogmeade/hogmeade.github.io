Cluster Routing Capsule Network Extension

  • Deep Learning
  • Pytorch
  • Capsule Network
  • Part-whole Relationship

A Capsule Network, or Capsule Neural Network (CapsNet), is designed to address limitations in traditional neural networks, focusing on hierarchical relationships and spatial hierarchies in data. Introduced by Geoffrey Hinton in 2017, the concept emphasizes dynamic routing for transparent decision-making in AI models, enhancing interpretability and fostering trust by revealing the decision-making process.

In contrast to dynamic routing, which involves iterative processes, cluster routing, a non-iterative method, outperforms with fewer parameters while preserving part-whole relationships. This work aims to clarify and analyze cluster routing, suggesting variations to achieve consensus among votes in distinct clusters. Exploring these variations can enhance cluster routing and capsule network capabilities, encouraging further exploration of this promising paradigm.
